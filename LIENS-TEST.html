📋 Étapes pour une partie test

  1. Ouvre Joueur 1 → Entre ton nom → Créer la salle
  2. Note le code à 6 lettres affiché
  3. Définis le budget (défaut : 2 500 or)
  4. Choisis un Général
  5. Ouvre Joueur 2 → Entre un nom → Colle le code → Rejoindre
  6. Joueur 2 choisit aussi un Général
  7. Joueur 1 (le host) clique "Lancer la bataille"
  8. Chaque joueur achète ses unités avec le budget
  9. Phase de déploiement : place tes unités dans la zone verte
  10. Clique "Prêt !" quand tu as placé ton Général
  11. La bataille commence ! ⚔️
⚠️ Si la version en ligne ne fonctionne pas : attends quelques minutes que CloudFront propage les mises à jour (invalidation en cours).
